# AI + Algorand RWA Verification

## 🚨 Problem
Real-world agreements (invoices, contracts) lack trust, verification, and transparency.

## 💡 Solution
We built an AI-powered verification system that validates agreements and records tamper-proof proof on Algorand blockchain.

## ⚙️ Features
- AI-based agreement verification
- Risk scoring system
- Wallet-based transaction confirmation
- Immutable blockchain storage

## 🧠 Tech Stack
- Frontend: React.js
- Backend: FastAPI
- Blockchain: Algorand (Testnet)
- Wallet: Pera Wallet
- SDK: algosdk

## 🔄 Workflow
Input → AI Verification → Risk Score → Wallet Confirmation → Blockchain Record

## 🔗 Key Highlight
Each verified agreement is permanently recorded on blockchain using wallet-signed transactions.

## 🎯 Use Case
SME invoice verification and trust-based transaction systems in India.
